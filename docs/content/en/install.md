## Installation

Add `audio-captcha` dependency to your project:

<code-group>
  <code-block label="Yarn" active>

```bash
yarn add audio-captcha
```

  </code-block>
  <code-block label="Pnpm">

```bash
pnpm add audio-captcha
```

  </code-block>
  <code-block label="NPM">

```bash
npm install --save audio-captcha
```

  </code-block>
</code-group>
